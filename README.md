# ShubraJewels

Royal Rajasthani jhumka store — **हर झुमका एक कहानी**.

Full-stack, fully admin-editable: Express + Mongoose (MongoDB Atlas) API,
React/Vite storefront with a Three.js 3D hero, and an admin panel with drag-drop
media uploads, a live theme editor, orders, and visitor analytics.

- **Run the API:** `cd server && npm install && npm run seed && npm run dev` (port 4200)
- **Run the web:** `npm install && npm run dev` (port 5273, proxies to the API)
- **Deploy:** see [DEPLOY.md](DEPLOY.md)
- **Architecture & conventions:** see [CLAUDE.md](CLAUDE.md)
